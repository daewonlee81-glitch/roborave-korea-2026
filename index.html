About

학생이 중심이 되는 로봇 경진의 무대

RoboRAVE Korea는 단순한 경쟁을 넘어, 학생들이 스스로 문제를 정의하고 해결하는 경험을 만듭니다.

RoboRAVE, 전 세계가 함께하는 로봇 축제

RoboRAVE는 미국에서 시작되어 현재 전 세계 60개국, 6,000여 팀의 학생들이 참가하는 국제 로봇경진대회입니다. 창의적 설계와 공학적 문제 해결을 종합적으로 평가하는 학생 주도형 프로젝트 기반의 경연입니다.

RoboRAVE Korea는 2026년부터 공식 출범하여 국내 참가팀 운영과 대회 기획, 한국 국가대표 선발을 담당하며, 학생들이 세계 무대에서 자신의 실력을 펼칠 수 있도록 지원합니다.

안내 · 본 규정은 RoboRAVE OSAKA 2026 세계대회 공식 규정을 RoboRAVE Korea가 한국어로 번역·정리한 자료입니다. 정식 규정은 2026년 6월 30일까지 일부 변경될 수 있으나, 로봇 제작 사양에는 영향을 주지 않습니다. 참가 구분: ES 초등 / MS 중등 / HS 고등 / UP 대학·일반.

01
a-MAZE-ing · 미로 탈출 Maze Navigation Challenge
경기 목적

높이가 있는 나무 미로(코스)에서 떨어지지 않고 자율주행 로봇으로 완주하는 것을 목표로 합니다. 제한 시간 내 완주 시 남은 시간에 비례한 보너스 점수가 가산됩니다.

참가 구분
초등 ES

2~4명으로 구성된 팀

로봇 요구사항
  • 자율형 — 외부 센서·리모컨 조작 모두 금지
  • 플랫폼 자유 (LEGO, Arduino, 자작 등)
  • 제작 비용 1,500 USD 이하
  • 바퀴 구성: 구동륜 2개 + 보조륜(캐스터) 1개 필수
  • 크기: 부피 65,030 ㎤ 이하
  • 센서 제한: 미로 추적용 외부 센서(적외선·초음파) 금지 / 휠 엔코더는 허용
일반 규칙
  • 경기 시간 120초(2분)
  • 시도 횟수·결선 진출(상위 8팀) 조건은 당일 이벤트 디렉터 결정
  • 연습 자유, 단 본 경기 시도가 우선
종목별 규칙
  • 코스 이탈: 바퀴가 하나라도 코스를 완전히 벗어나거나 받침 지면에 닿으면 탈락
  • 재도전: 시간 내라면 출발선으로 돌아가 횟수 제한 없이 재도전 가능
코스 사양
  • 재질: 파티클보드 또는 합판 (폭 24cm, 높이 2cm)
  • 형상: 직선 구간 + −45°, −90°, −135° 코너 조합
  • ES 구성: 직선 4구간 + 코너 3개 (총 500점 만점)
채점

앞바퀴가 각 구간의 채점 라인에 닿을 때 점수가 가산됩니다.

항목점수
직선 구간 완료50점
코너 완료100점
타임 보너스잔여 1초당 1점 (소수점 절사)
  • 중도 종료: 시간 내 완주하지 못한 경우 통과한 가장 먼 구간까지 점수 기록
  • 완주 시: 부문 만점 + 타임 보너스
02
SumoBot (Lego & VEX IQ) · 스모봇 (레고/VEX IQ) SumoBot — LEGO & VEX IQ Platform Only
경기 목적

자율형 로봇을 제작해 도효(링) 위에서 상대 로봇을 탐색·밀어내 링 밖으로 떨어뜨리는 경기입니다.

참가 구분 / 중량 제한
초등 ES
  • LEGO 또는 VEX IQ 전용: 1kg 이하
  • ※ 등록이 5팀 미만이면 부문이 통합될 수 있음
로봇 요구사항
  • 자율형 (원격조작 금지)
  • 비용 1,500 USD 이하
  • 크기: 최대 25cm × 18cm (높이 제한 없음 · 시작 시 측정)
  • 가동 파츠 허용되나 상기 크기 내에 수납되어야 함
  • 구조 제한: 망치·회전톱 등 상대 손상 의도 공격 금지 (스쿱·슬라이딩 플레이트는 허용)
  • 특칙: 순정 LEGO 또는 VEX IQ 부품만 사용
링 사양
  • 형상: 지름 100cm 원형 (흑/백 바닥)
  • 가장자리: 폭 5cm 경계선 (흑/백 결정은 당일 발표)
  • 높이: 바닥에서 50~80mm (낙하 = 패배 룰을 위한 단차)
경기 규칙
  • 형식: 예선 라운드로빈
  • 1매치 2판 선승제
  • 승점: 승리 3점 / 무승부 1점 / 패배 0점
  • 시작: 링 반대편 흰 라인에 닿게 배치(방향 자유), 시작 후 3초 대기
  • 판정: 먼저 바닥에 닿은 쪽이 패. 60초 무승부 시 인정
  • 로봇끼리 정체 5초 시 재시작 가능
  • 1매치 최대 5분
  • 심판은 동일 동작 반복 시 시간 정지 없이 재시작 명령 가능
03
Open SumoBot · 오픈 스모봇 Open Platform SumoBot
경기 목적

자율형 로봇을 자유 플랫폼으로 제작해 도효에서 상대 로봇을 탐색·밀어내는 경기.

부문 / 중량 제한
초등 ES 중등 MS 고등 HS 대학·일반 UP
중량ESMSHSUP
1 kg◯◯◯◯
3 kg−◯◯◯

※ 5팀 미만 부문은 통합 가능

로봇 요구사항
  • 자율형 (모든 플랫폼 가능, 원격조작 금지)
  • 비용 1,500 USD 이하
  • 크기: 최대 25cm × 18cm (높이 제한 없음)
  • 금지: 망치·회전 커터 등 의도적 파괴 장치 (스쿱·슬라이딩 플레이트는 허용)
링 사양
  • 지름 100cm 원형, 가장자리 5cm 경계선, 바닥에서 50~80mm
경기 규칙
  • 예선 라운드로빈, 1매치 2판 선승제
  • 승점: 승 3 / 무 1 / 패 0
  • 1매치 3분
  • 60초간 움직임 없거나 5초간 락(lock) 상태 → 심판 판단으로 무승부/재시작
⚠️ 중요: 로봇 기동 프로토콜
이 규칙은 매우 중요합니다. 위반 시 매치 결과에 직접 영향을 줄 수 있습니다.
  • 아레나 진입: 로봇을 기동할 수 있는 팀원은 1명만 가능
  • 시작 위치: 심판이 지정한 마커 위치에 배치
  • 방향: 로봇 "정면(표시한 면)"이 조작자 자신을 향하도록 링 가장자리에 배치
  • 카운트다운: 준비 완료 신호 후 심판의 3초 카운트다운으로 시작
  • 필수 동작: 기동 직후, 다른 동작(공격 등) 전에 반드시 그 자리에서 1회전(360° 이상)
"낙하" 정의
  • 패배: 로봇 일부가 조금이라도 바닥(도효 받침 면)에 닿으면 낙하
  • 계속: 링 밖으로 튀어나와 있어도 바닥에 닿지 않으면 경기 진행
04
Line Following (Lego & VEX IQ) · 라인트레이싱 (레고/VEX IQ) Line Following — LEGO & VEX IQ Platform Only
경기 목적

자율형 로봇으로 흰 바탕의 코스 위 검은 선을 따라가 타워까지 이동, 공을 전달합니다. 1왕복으로 "완주" 인정 후, 제한 시간 내 부문별 "정확한 개수"의 공을 추가 운반.

참가 구분
초등 ES
로봇 요구사항
  • 자율형 (원격 조작 금지)
  • 접수 시 라인트레이스 + 타워 정지 실연 필요
  • 다중 센서·프로세서 사용 가능
  • 크기: 시작 시 부피 65,030 cm³ 이하
  • 비용 1,500 USD 이하
  • 특칙: 순정 LEGO 또는 VEX IQ 부품만 사용
코스 / 타워 사양

타워: 높이 20cm × 폭 10cm × 길이 35cm. 상부에 10cm 정사각 투입구, 뒷면은 공이 통과하도록 개방.

주의 · 사람이 타워에 닿거나 안을 들여다보면 실격(일률 400점). (기본 주행 완료 시 적용되며, 공 점수가 무효 처리됩니다.)

트랙: 120cm × 180cm. 교차로 없음, 라인 폭 1.25cm. 곡선 반경 15cm 이상, 타워 직전 20cm는 직선. 자연광 영향이 있을 수 있어 센서 캘리브레이션 필요.

STEP 1: 기본 주행 (1왕복) — 최대 400점

출발 → 타워에서 최소 1개 공 전달 → 출발지 복귀.

부문출발타워 정지공 전달복귀
ES50100100150

  • 로봇 조작자만 로봇·편집 기기(노트북·태블릿 등)를 접촉 가능. 경기 중 비교 구역 외 반출 및 통신 소프트웨어 사용 불가
  • 운영 원칙: 조작자가 실행하고, 코치는 과도한 개입 없이 지도, 관전자는 관전 예의를 지킬 것
대회 방식
  • 연합대(Alliance) 조별 또는 팀 간 대결. 연합대 구성은 대회 측이 무작위 배정 후 공고
  • 각 팀의 경기 수는 접수 마감 후 사전 공고
  • 각 경기 후 적분(Tournament Points)과 득점(Score) 2종 기록 부여
  • 득점: 해당 경기에서 소속 연합대가 획득한 점수
  • 적분: 승리 연합대 3점 / 무승부 2점 / 패배 연합대 1점
  • 최종 승부는 전 경기 적분 합계로 결정
동점 시 우선순위
순위판단 기준승부

경기 규칙
  • 연합대 또는 팀 간 대결. 연합대 방식이면 A·B 연합대 각각 2개 팀 무작위 구성, 원격조종 방식으로 진행
  • 빨강·파랑 베이스 구역이 출발 위치. 출발 시 차체의 어느 한 접지점이 베이스 색상 블록에 닿아 있어야 함. 로봇 위에 미리 블록을 보유 불가
  • 원격조종으로 자기 팀 색상 블록을 잡아 득점 구역에 배치(구역 밖 돌출 금지). 1개 블록당 10점
  • 배가(倍加) 버튼: 득점 구역의 배가 버튼을 눌러 LED 띠 색을 자기 팀 색으로 설정 시, 해당 구역의 같은 색 블록 득점이 2배 가산
  • 중앙 특수 득점 구역: 중앙에는 위로 블록을 쌓을 수 있으며 (최대 4개), 경기 종료 시 가장 위 블록의 색상 팀이 그 구역 전체 블록 개수 × 2배 점수 획득
  • 블록 장외 이동: 득점 블록이 경기장 밖으로 나가면 이동한 쪽 반칙, 블록당 40점 감점 (누적)
수리 / 정비
  • 로봇 수리 필요 시 베이스 구역까지 자력 복귀 후 차체 접지점이 베이스 색상 블록에 닿으면 선수가 직접 반출 가능. 자력 복귀 불가 시 심판 도움 요청
  • 수리 중에는 기존 구조 수리·조정만 허용. 추가 부품 부착 불가 (위반 시 원상복구 후 재경기)
  • 수리 완료 후 베이스 구역에 다시 배치하고 출발. 차체 접지점이 베이스 블록과 접촉해야 함
  • 수리 시 로봇 위 블록은 임의 위치로 경기장에 무작위 재배치
경기 종료 시
  • 로봇이 자기 베이스 구역(접지점)에 복귀해 있으면 로봇당 추가 20점 보너스
  • 심판이 접촉 허용을 선언하기 전 임의 이동 시: 베이스 안에 있던 로봇이 이동으로 벗어나면 보너스 취소 / 베이스 밖에 있던 로봇이 이동으로 들어가도 보너스 미지급
  • 경기 중 로봇이 행위 능력을 상실하고 5초 내 정상 동작 회복 못 하면 심판이 장외 이동, 경기 속행
반칙 행위 (각 항목 누적)
  • 외부에서 베이스 구역에 배치 후 출발 전 규정 미준수
  • 고의로 상대 베이스 왕복을 방해 (심판 인정)
  • 로봇이 능동적으로 상대 로봇과 충돌하거나 방해 시도 (심판 인정)
  • 고의 충돌·밀치기: 정지한 상대 로봇을 강제로 이동시키거나, 자기 전진 동작으로 상대를 강제로 후퇴시키는 행위
  • 경기 도구·시설을 임의로 파손 (심판 인정)
  • 악성 충돌: 40cm 이상 거리에서 가속 후 그대로 충돌, 또는 20cm 내에서 전후 진동으로 3회 이상 부딪힘
  • 경기 개시 후 종료 전, 비교 구역 내에서 선수가 경기 중인 로봇에 접촉 (심판 허용 시 제외)
  • 수리 시 정비 구역으로 원격 이동 후 반출하지 않고 정비 구역에서 직접 수리
  • 경기 종료 후 심판의 회수 허가 전 임의 이동

"축구"를 목표로 2대의 원격조종(Radio-controlled) 로봇을 설계·제작·프로그래밍합니다.

로봇 요구사항

플랫폼 자유. 2대 1조의 원격조종 로봇을 사용하며 비용은 500 USD 이하. 접수 시 사양 검사 진행.

  • 최대 크기: 15cm × 15cm × 15cm. 허용 오차 없음
  • 크기·중량 제한은 대회 전 기간에 걸쳐 엄격 적용
  • 상대 로봇과 접촉할 가능성이 있는 부품은 쉽게 파손되지 않을 만큼 견고할 것
  • 금지 메커니즘: 슈팅(shooting), 잡기(grabbing), 킥(kicking), 클램핑(clamping), 점착성 소재, 솔레노이드, 공을 밀거나 잡는 모든 시스템 금지. 단, 경기 시작 후 부품을 전개하는 것은 허용
  • 전원: 배터리 사용. 외부 케이블 전원 공급 불가
  • 공 포함 한계: 로봇 하나당 공 부피의 최대 30%만 감쌀 수 있음. 나머지 70%는 외부에 노출되어야 함
    • 공: 지름 35.5mm, 무게 약 40g, 30% 기준 10.65mm
    • 공정 경기를 위해 정밀 측정·중량 검사 시행. 사양 미준수 로봇은 출전 불가
  • 접착제, 흡착, 점착 장치로 공을 로봇에 부착하는 행위 엄격 금지
  • 로봇에는 팀명과 번호를 명확히 표시. 팀 간 로봇 공유 불가
일반 규칙 — 페어 플레이 위반 (상대팀에 1골 부여)
  • 경기 중단 후 재개까지 30초 초과
  • 대회의 무결성을 훼손하는 발언·행동 → 해당 팀 실격
  • 상대 팀 로봇을 의도적으로 조종 → 실격
  • 경기 중이 아닌 팀이나 관중이 의도적 방해 시, 이로부터 이익을 본 팀 실격
  • 인화성 장치 사용 → 실격
  • 필드 또는 행사 설비 고의 손상 → 실격
  • 심판·상대 팀 모욕 → 실격
대회 방식
  • 조 편성: 무작위 추첨 또는 알파벳 순 (참가팀 수에 따라 결정)
  • 각 조는 4팀으로 구성
  • 조별 라운드로빈, 1경기 최대 3분
  • 각 조 상위 2팀이 토너먼트(단판 토너먼트) 진출
경기 시간
  • 1경기 최대 3분. 전·후반 각 90초로 진행
  • 전반 종료 후 30초 이내로 좌우 교체
  • 연장·타임아웃·로스타임 없음
  • 공이 필드를 벗어나면 시간 정지, 공이 다시 들어오면 시간 재개
승점 배분
  • 득점 우세 팀 → 3점
  • 상대 팀이 파손·배터리 소진으로 경기 속행 불가능 → 비록 상대가 앞서고 있었어도 3점
  • 부전승 (상대 미출전): 승자에게 3점 + 3골 부여
  • 득점 없는 무승부 또는 동점 무승부 → 양 팀 1점
  • 양 팀 모두 속행 불가능 (경기 진행 중이었던 경우) → 0점
  • 패배 팀 → 0점
  • 양 팀 모두 미출전 → 0점
동점 처리
  • 조별 라운드: 동점 시 총 득점이 많은 팀 승. 그래도 동점이면 코인 토스
  • 토너먼트 라운드: 동점 시 경기 속행하여 골든 골로 결정
토너먼트 대진 (8강 기준)

1위 vs 8위, 4위 vs 5위, 3위 vs 6위, 2위 vs 7위 → 준결승(S1~S4) → 결승(F1·F2) → 우승

필드 / 트랙
  • 트랙은 일반적으로 내구성 종이 / PVC 비닐 백킹 / 3mm Trovicel에 인쇄 (120cm × 180cm)
  • 필드 크기: 1.20m × 1.80m
  • 골 크기: 0.50m
